BipedalLocomotion::IK::AngularMomentumTask class
#include <BipedalLocomotion/IK/AngularMomentumTask.h>

AngularMomentumTask is a concrete implementation of the Task.

Contents

Please use this element if you want to control robot centroidal angular momentum. The task assumes perfect control of the robot velocity that contains the base linear and angular velocity expressed in mixed representation and the joint velocities. The task represents the following equation

where is the angular part of the robot centroidal momentum matrix is the desired centroidal angular momentum.

Note

AngularMomentumTask can be used to control also a subset of element of the AngularMomentum. Please refer to mask parameter in IK::AngularMomentumTask::initialize method.

PolyDriverDescriptor constructMultipleAnalogSensorsClient(std::weak_ptr<const BipedalLocomotion::ParametersHandler::IParametersHandler> handler)

Helper function that can be used to build a MultipleAnalogSensorsClient device.

Parameters
handler pointer to a parameter handler interface.
Returns A PolyDriverDescriptor. In case of error an invalid PolyDriverDescriptor is returned.

Note

The following parameters are taken into consideration

Parameter NameTypeDescriptionMandatory
descriptionstringDescription of the multiple analog sensor client. It is the device nameYes
remote_port_namestringName of the port associate to the remoteYes
local_prefixstringPrefix of the local port (e.g. the application name)Yes
local_port_name_postfixstringPostfix of the local port. The local port name is /<local_prefix><local_port_name_postfix>Yes
timeoutdoubleTimeout in seconds after which the device reports an error if no measurement was received. (Default value 0.01)No
external_connectionboolIf set to true, the connection to the rpc port of the MAS server is skipped and it is possible to connect to the data source externally after being opened. (Default value false)No
carrierstringThe carrier used for the connection with the server. (Default value tcp)No

The MultipleAnalogSensorsClient device is implement in yarp.

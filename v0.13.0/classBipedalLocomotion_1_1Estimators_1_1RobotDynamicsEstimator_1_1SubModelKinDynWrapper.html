BipedalLocomotion::Estimators::RobotDynamicsEstimator::SubModelKinDynWrapper class
#include <BipedalLocomotion/RobotDynamicsEstimator/SubModelKinDynWrapper.h>

SubModelKinDynWrapper is a concrete class and implements a wrapper of the KinDynComputation class from iDynTree.

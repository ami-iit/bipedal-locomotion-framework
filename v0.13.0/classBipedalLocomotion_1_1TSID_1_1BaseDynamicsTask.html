bool BipedalLocomotion::TSID::BaseDynamicsTask::setVariablesHandler(const System::VariablesHandler& variablesHandler) override

Set the set of variables required by the task.

Parameters
variablesHandler reference to a variables handler.
Returns True in case of success, false otherwise.

The variables are stored in the System::VariablesHandler.

Note

The handler must contain variables named as the parameter robot_acceleration_variable_name, variable_name (in CONTACT_<i> group) and stored in the parameter handler. robot_acceleration_variable_name represents the generalized acceleration of the robot. Where the generalized robot acceleration is a vector containing the base spatial-acceleration (expressed in mixed representation) and the joints acceleration.

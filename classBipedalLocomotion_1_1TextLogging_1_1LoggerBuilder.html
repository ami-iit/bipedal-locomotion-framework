BipedalLocomotion::TextLogging::LoggerBuilder class
#include <BipedalLocomotion/TextLogging/LoggerBuilder.h>

LoggerBuilder is a class that implements the Builder paradigm.

Contents

  • Reference
    • Public static functions
    • Friends

You can use the LoggerBuilder at the beginning of your application to set a different sink for spdlog. For instance, you can use the following example to enable the TextLogging with YARP

#include <BipedalLocomotion/TextLogging/Logger.h>
#include <BipedalLocomotion/TextLogging/YarpLogger.h>
#include <BipedalLocomotion/TextLogging/LoggerBuilder.h>

// Change the logger
BipedalLocomotion::TextLogging::LoggerBuilder::setFactory(std::make_shared<BipedalLocomotion::TextLogging::YarpLoggerFactory>());

BipedalLocomotion::log()->info("My info");

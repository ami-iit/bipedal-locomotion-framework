BipedalLocomotion::IK::IntegrationBasedIKProblem struct
#include <BipedalLocomotion/IK/IntegrationBasedIK.h>

IntegrationBasedIKProblem store all the ingredients to run an modfy at runtime an Integration based inverse kinematics problem.

Template parameters
index a positive number from 0 to 2.

Note

0 is associated to the variablesHandler, 1 to the weights and 2 to the ik. Thanks to this method the IntegrationBasedIKProblem behaves as a tuple. I.e., the std::tie() or Structured binding declaration are supported by IntegrationBasedIKProblem.
